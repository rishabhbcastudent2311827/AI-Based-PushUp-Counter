# 💪 AI Push-Up Counter

## 📌 Project Overview
AI Push-Up Counter is a desktop-based fitness application that automatically counts push-ups using computer vision.  
The system uses a webcam to detect body posture and calculate push-ups in real-time with high accuracy.

---

## 🚀 Features
✔ User Login & Registration System  
✔ Real-time Push-Up Detection using AI  
✔ Automatic Push-Up Counting  
✔ Calories Burn Calculation  
✔ Workout History Tracking  
✔ Secure Password Storage (Hashing + Salt)  

---

## 🧠 Technology Used
- Python  
- OpenCV  
- MediaPipe  
- Tkinter (GUI)  
- SQLite (Database)  

---

## ⚙️ How It Works
1. User logs into the system  
2. Clicks on **Start** to begin workout  
3. Camera detects body posture using AI  
4. Push-ups are counted automatically  
5. Calories are calculated  
6. Data is saved into database  
7. User can view workout history
